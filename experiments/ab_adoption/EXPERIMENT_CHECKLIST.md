# Experiment Checklist

- Fresh session started
- Same model and version used across conditions
- Same prompt used across conditions
- Same starting repository state used across conditions
- Same task used across conditions
- Same temperature or comparable settings used when configurable
- Timing started before the agent task began
- Timing stopped after the final artifact or run log was written
- `timed_run.py --manual` used for interactive Codex sessions when needed
- Elapsed time not edited by hand unless uncertainty was clearly noted
- Results logged
- Notes recorded
- Trial workspace reset from clean template
